# Frisbee Golf Server

## Handicap System
It will use a slightly modified golf handicap system, to allow for handicaps to be produced with fewer games, but will be the same after many games. It will use the best x/2 games of your last x games, where x < 20.

## Database scheme
--------------------------------------------------------
| USERS           | Pid  | Name   | Grid       |       |
--------------------------------------------------------
| GAMES           | Gid  | Cid    | Time       | Grid  |
--------------------------------------------------------
| PLAYER_GAMES    | Pid  | Gid    | Scores[]   |       |
--------------------------------------------------------
| GROUPS          | Grid | Name   | Passwords  |       |
--------------------------------------------------------
| COURSES         | Cid  | Pars[] | Name       | Grid  |
--------------------------------------------------------

## Get Requests
/group/players/<Grid>   >> Pid[]
/group/courses/<Grid>   >> Cid[]
/group/<name>           >> Grid
/group                  >> Grid[]
/player/<pid>           >> {Name, Handicap, Grid, Gid[]}
/games/<gid>            >> {Pid[], Scores[], Cid, time}
/course/<cid>           >> {Name, Pars[]}

## post Requests
/newgame/<Grid>         << {Pid[], Scores[], Cid}
/newcourse/<Grid>       << {Pars[], Name}
/newplayer/<Grid>       << {Name}
